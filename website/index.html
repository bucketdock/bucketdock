Built for people who actually use S3.

Connections live next to your work. Transfers stay out of your way. Secrets stay in your Keychain.

☁️

Multi-provider

AWS S3, Cloudflare R2, MinIO, Backblaze B2 and any S3-compatible endpoint. Save as many connection profiles as you need.

📦

Bucket browser

Folder-style navigation, breadcrumbs, multi-select, context menus, drag-and-drop upload and recursive download.

🚀

Transfer queue

A bottom-right dock tracks every upload, download, and copy. Cancel running transfers and retry the failures.

↔️

Bucket-to-bucket copy

Move objects between buckets, even across providers. Pick a destination connection, bucket and prefix.

🔐

Keychain-backed secrets

Access keys live in the macOS Keychain under com.bucketdock.app. Never in plaintext, never in browser storage.

🦀

Rust backend

All S3 traffic is signed by the Rust AWS SDK in the desktop process. The UI never speaks S3 directly from the browser runtime.

🗂

Object metadata

Inspect and edit Content-Type, Cache-Control, Content-Disposition and custom user metadata in place.

🌗

Native feel

Hidden titlebar, native file pickers, light and dark mode, persistent sidebar width and selection state.
